Help: xRoster

Description

Simple application and roster page for people to be listed who are in your club / group, etc.

Install/uninstall

No special measures necessary, follow the standard installation process – extract the /xroster folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions

To set up this module you need to:

  • Configure your preferences for the module (see ‘Preferences’) and block if you intend to use it (see ‘Blocks’).
  • Edit existing Categories, Groups, and Titles, and Members, or add new ones.
